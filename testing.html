10. Menggunakan CSS Framework Online (CDN)

CDN = Content Delivery Network - Kita bisa langsung pakai CSS yang sudah dibuat orang lain tanpa download!

A. Bootstrap - Framework CSS Paling Populer

Cara pakai: Tambahkan link ini di <head> tag:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

Contoh Button Bootstrap:

*Button di atas dibuat manual, tapi dengan Bootstrap cukup tulis: <button class="btn btn-primary">


B. Tailwind CSS - Utility-First Framework

Cara pakai: Tambahkan link ini di <head> tag:

<script src="https://cdn.tailwindcss.com"></script>

Contoh dengan Tailwind:

Dengan Tailwind: class="bg-blue-500 text-white p-4 rounded-lg"

Cara pakai: Tambahkan link ini di <head> tag:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

Cara pakai: Tambahkan link ini di <head> tag:

    <meta charset="UTF-8">

    <!-- Link Bootstrap CSS dari CDN -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" 
          rel="stylesheet">

    <div class="container mt-5">
        <h1 class="text-primary">Hello Bootstrap!</h1>
        <button class="btn btn-success">Click Me</button>
        
        <div class="alert alert-info mt-3">
            Ini adalah alert box dari Bootstrap!
        </div>
    </div>
</body>
</html>

Tips:

  • Inline CSS: Langsung di element (style="...")
  • Internal CSS: Di dalam tag <style> di <head>
  • External CSS: File .css terpisah (paling recommended)
  • Class: Gunakan untuk styling yang dipakai berkali-kali (.classname)
  • ID: Gunakan untuk styling element unik (#idname)
  • CSS Framework: Pakai Bootstrap/Tailwind untuk project cepat dan profesional!
